# 🩺 Diabetes Prediction using Machine Learning

## 📌 Project Overview
This project aims to predict whether a person has diabetes based on medical attributes using machine learning techniques. The model helps in early detection and analysis of diabetes.

---

## 🚀 Features
- Data preprocessing and cleaning
- Handling missing/invalid values
- Feature scaling using StandardScaler
- Model training using Random Forest Classifier
- Model evaluation using multiple metrics

---

## 📊 Dataset
- Pima Indians Diabetes Dataset
- Contains medical attributes such as Glucose, BMI, Age, etc.
- Target variable: Outcome (0 = Non-Diabetic, 1 = Diabetic)

---

## 🛠️ Technologies Used
- Python
- pandas
- NumPy
- scikit-learn
- Matplotlib
- Seaborn

## ⚙️ Project Workflow
1. Data Collection

6. Model Evaluation  
7. Prediction
